# ATELA Marketing Site — UI Kit

High-fidelity recreation of the ATELA marketing landing page. Based on Superside's structural rhythm (oversized display type, dense social proof, full-bleed editorial imagery) with ATELA's ink + lime identity.

## Files
- `index.html` — interactive landing page
- `Nav.jsx`, `Hero.jsx`, `LogoCloud.jsx`, `Features.jsx`, `BeforeAfter.jsx`, `Stats.jsx`, `CaseStudy.jsx`, `Pricing.jsx`, `FinalCta.jsx`, `Footer.jsx`

## Tweaks exposed
- Hero variant (split image / full-bleed / typographic)
- Accent color (lime / magenta / violet)
- Dark vs light hero
